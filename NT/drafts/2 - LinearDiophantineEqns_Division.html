Linear Diophantine Equations and Division

11/21/2016

The basic object of study in Number Theory are the integers, the set of numbers $...,-2,-1,0,1,2,...$, which we denote with the symbol $\mathbb{Z}$, along with its arithmetic operations of addition and multiplication. These operations satisfy associative and distributive properties: $$ \begin{eqnarray*} a+(b+c) &=& (a+b)+c\\ a\cdot(b\cdot c) &=& (a\cdot b)\cdot c\\ a\cdot (b+c) &=& (a\cdot b)+(a\cdot c) \end{eqnarray*} $$ We will just take these basic properties of arithmetic for granted. Number Theoretic questions begin to naturally pop-out when you ask certain kinds of questions about the integers. In particular, questions about Diophantine Equations motivate many results in Number Theory and are the subjects of the hardest unanswered questions in the whole of math. A Diophantine Equation is an algebraic equation for which you only look for integer solutions. For example: $$ \begin{eqnarray*} x^2+y^2=z^2 && x^2-7y^2=1\\ x^7+y^7 +z^7+w^7=0 && y^2=x^3+x+1 \end{eqnarray*} $$ are all Diophantine Equations when you are searching only for the solutions that are integers. For example, the equation $x^2-2=0$ has an easy solution in the reals: $x=\pm\sqrt{2}$, but it takes some work to show that this has no solution as a Diophantine Equation, as $\sqrt{2}$ is irrational.

Diophantine Equations are also responsible for much of our arithmetic. To this end, we will study Single Variable Linear Diophantine Equations and see how asking questions about these naturally leads to division and fundamental results about division. A Single Variable Linear Diophantine Equation is an equation of the form $ax-b=0$ for two fixed integer $a,b\in\mathbb{Z}$. We seek to know when this has an integer solution. To this end we define:

Definition: Let $a,b\in\mathbb{Z}$ be integers with $a$ nonzero. Say that $a$ Divides $b$ if there is a solution to the Diophantine Equation $ax-b=0$. Write $a|b$ for this, read "$a$ divides $b$". We might also say that $b$ is a Multiple of $a$. If $a$ does not divide $b$, write $a\nmid b$.
All we have done is provide ourselves with a streamlined way to talk about when $ax-b=0$ has a solution. We can then use the form of this equation to provide some structure to the solutions via the following theorem.
Theorem: Let $a,b,c\in \mathbb{Z}$ be integers, with $a$ nonzero. We have the following results about divisibility:
  1. For all $c$ we have $c|c$, $1|c$ and $c|0$, even if $c=0$.
  2. If $a|b$ and $b\ne 0$ then $|a|\leq |b|$.
  3. If $a|b$ then $a|bc$.
  4. If $a|b$ and $a|c$ then $a|(b+c)$.
  5. If $a|b$ and $b|c$ then $a|c$.
  6. HW: If $|c| < |a|$ and $a|b$, then $a\nmid (b+c)$.
  7. HW: If $a|b$ and $b|a$ then $a=\pm b$.
Proof:
  1. If $c$ is any integer, then $x=1$ is a solution to $cx-c=0$, and $x=c$ is a solution to $1\cdot x-c=0$, and $x=0$ is a solution to $cx-0=0$.
  2. Let $x=n$ be a solution to $ax-b=0$. Then $b=an$. If $b\ne 0$, then $n\ne 0$ and $|n|\geq 1$, so $|b|=|an|\geq |a|$.
  3. Let $x=n$ be a solution to $ax-b=0$ and $x=m$ a solution to $ax-c=0$. We then have $a(n+m)-(b+c)=an-b + am-c = 0+0$.
  4. Let $x=n$ be a solution to $ax-b=0$ and $x=m$ a solution to $bx-c=0$. Then $anm -c=bn-c=0$.
This gives us some basic rules to work with divisibility and allows us to discuss solutions to these kinds of equations. Furthermore, these equations allow us to derive the arithmetic of fractions from their solutions.
Theorem: If $a,b\in\mathbb{Z}$ are integers, with $a$ nonzero and $a|b$, then let the symbol $x=\frac{b}{a}$ denote the solution to the equation $ax-b=0$. We then have the following arithmetic rules:
  1. For integers $a,b,c$, with $a,c$ nonzero, we have $\frac{bc}{ac}=\frac{b}{a}$.
  2. For integers $a,b,c,d$ with $a,c$ nonzero, we have $\frac{b}{a}\cdot \frac{d}{c}=\frac{bd}{ac}$
  3. For integera $a,b,c$, with $a$ nonzero, we have $c\cdot \frac{b}{a}=\frac{cb}{a}$
  4. For integers $a,b,c,d$ with $a,c$ nonzero, we have $\frac{b}{a}+ \frac{d}{c}=\frac{bc+ad}{ac}$
  5. For an integer $b$ we have $\frac{b}{1}=b$ for all $b$ and $\frac{0}{b}=0$, $\frac{b}{b}=1$, when $b\ne 0$
Of course we know that these symbols are fractions, but we haven't proved that yet, these are just symbols to represent particular integer solutions to particular equations. We can then prove basic results about how these objects combine and interact, and will discover fractions. So in the following proof, we cannot assume anything about fractions because we don't know that these are fractions. In fact, this is a way to actually construct the rational numbers, as all possibly symbols of this kind, even when a solution does not exist, and impose that these rules hold.
Proof: The basic idea to these is that each equation $ax-b=0$ has exactly one solution, so if we have a number $N$ and find that $aN-b=0$, that is $x=N$ is a solution, then we must conclude that $N=\frac{b}{a}$. So we do need to show that each of these equations has only one solution. If $x=N,M$ are solutions to $ax-b=0$ then $aN=b=aM$. Since $a$ is nonzero, we can cancel to get $N=M$.
  1. We just need to show that $x=\frac{bc}{ac}$ is a solution to $ax-b=0$. By definition of this symbol, we have $ac\cdot\frac{bc}{ac}-bc=0$, from which we can pull out a $c$ to get $c(a\frac{bc}{ac}-b)=0$. Since $c\ne 0$, it follows that we must have $a\frac{bc}{ac}-b=0$, which means that $x=\frac{bc}{ac}$ is a solution to $ax-b=0$, which implies (as mentioned above) that $\frac{bc}{ac}=\frac{b}{a}$.
