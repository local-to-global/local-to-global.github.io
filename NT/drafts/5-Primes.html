Prime Numbers

11/21/2016

Last time we briefly discussed coprimality and saw how they represent a lack of "multiplicative overlap". This can be used to simplify and understand many results and concepts going forward, and is part of our basic vocabulary. We now discuss a type of number that is one of the most studied, yet mysterious objects in number theory: Prime numbers. The idea is that coprimality is a very nice property that two numbers can have, so numbers that are coprime to as many numbers as possible would be really nice to work with. This is the basic idea about primes: They are coprime to as many numbers as they possible can be. They have really nice properties and have shaped how we study object in number theory by allowing us to break up hard problems into many smaller, easier problems associated to nothing more than a single prime. They are defined by the following thoerem:

Theorem: Let $p$ be a nonzero number, not equal to $\pm 1$ and $a,b$ any integers. The following are equivalent:
  1. If $p|ab$ then $p|a$ or $p|b$
  2. $a|p$ if and only if $|a|=1$ or $|a|=|p|$
  3. If $0 < |a|<|p|$ then $GCD(a,p)=1$
  4. If $0<|a|<|p|$ then $ax+py=1$ has a solution
  5. If $GCD(a,p)>1$ then $p|a$
Proof: Some of these are fairly obvious, but are good to be stated for reference and clarity.
  • $(1\rightarrow 2)$ It is clear that $a=1,-1,p,-p$ all divide $p$, regardless of what $p$ is. So assume that $a|p$ so that $p=ac$ for some integer $c$. This means that $p|ac$ and so $p|a$ or $p|c$. If $p|c$, then $c=dp$ and we can write $p=adp$, which means that $ad=1$ and this can only happen if $a=\pm 1$. Now assume that $p|a$ and write $a=pd$. This means that $p=pdc$ and so we need $dc=1$ which can only happen if $|d|=1$ and so $|a|=|p|$.
  • $(2\rightarrow 3)$ Assume that $|a|<|p|$ and $d=GCD(a,p)$. Since $d|p$, it follows that $d=1$ or $d=p$. If $d=1$ then we're done, so assume that $d=p$. In this case $p|a$ and so $|p|\leq |a|$, which can't happen. So $d=1$.
  • $(3\leftrightarrow 4)$ The case 4 is equivalent to 4 directly from Bezout's Identity.
  • $(4\rightarrow 5)$ Assume that $d=GCD(a,p)>1$. Write $a=pq+r$ for $0\leq r < |p|$. From our previous result, we have $d=GCD(r,p)$. Since $r< |p|$, if $r\ne 0$ then $d=1$ by assumption, but this is not the case. The only way for $d>1$ is if $r=0$ in which case $d=p$ and so $p|a$.
  • $(5\rightarrow 6)$ Assume that $p|ab$ and that $p\nmid a$. By assumption, we must have $GCD(a,p)=1$ and so there are some $x,y$ so that $px+ay=1$. Assume that $GCD(p,b)=1$, and so there is some $s,t$ so that $ps+bt=1$. This implies that we have $$ 1 = (px+ay)(ps+bt) = p(pxs+xbt+ays) +ab(ty) $$ which means that there is a solution to $pX+abY=1$ which implies that $GCD(p,ab)=1$, which is not the case since $GCD(p,ab)=|p|$. It then follows that $GCD(p,b)>1$ and so, by our assumption, $p|b$.

We say that a number that satisfies the above equivalent conditions is Prime.
