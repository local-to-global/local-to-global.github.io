A Derivation of the Euler-Maclaurin Formula

11/21/2016

The Euler-Maclaurin Formula is a formula that allows for the approximation of sums by integrals to good asymptotic accuracy. It the backbone of many proofs, including Stirling's Approximation and many from Analytic Number Theory. From the derivation, it is not clear why Bernoulli Numbers and Bernoulli Polynomials appear, these are objects with close ties to Number Theory, but are seen in this approximation theorem. We'll give a proof that investigates these matters.

Bernoulli-type Functions

Bernoulli polynomials and numbers are typically given as coefficients in the power series of a function with particular properties. Let's define a type of function that behaves like these, and see how we can look at the Euler-Maclaurin Fromula in this context. Define a Bernoulli-type Function as a bivariate function $f(x,t)$ that has a power series in the variable $t$: $$ f(x,t) = \sum_{k=0}^{\infty}f_k(x)\frac{t^k}{k!} $$ Further, assume that it satisfies $\frac{d}{dx}f(x,t)=tf(x,t)$. With a Bernoulli-type function $f(x,t)$ and another functio $g(x)$, we can define the integral transformation of $g(x)$ over some interval using $f(x,t)$ as a kernel. That is, set: $$ [g]_f(t) = \int_a^bf(x,t)g(x)dx $$ Integration by parts then gives us the formula: $$ t[g]_f(t) = f(b,t)g(b)-f(a,t)g(a) - [g']_f(t) $$ This formula contains the pertinent information of the Euler-Maclaurin Formula. For an arbitrary function $h(x)$ we can write $$ [h]_f(t) = \sum_{k=0}^{\infty}\frac{t^k}{k!}\int_a^b f_k(x)h(x)dx $$ We can then compare the coefficients in the main theorem using this. The coefficient of the constant term gives $\int_{a}^b g'(x)f_0(x)dx = f_0(b)g(b)-f_0(a)g(a)$. For $k>0$, we get $$ k\int_a^b f_{k-1}(x)g(x)dx = f_k(b)g(b)-f_k(a)g(a) - \int_a^bf_k(x)g'(x)dx $$ If we iterate this process, we can get the formula $$ t^{n+1}[g]_f(t) +(-1)^{n} [g^{(n+1)}]_f(t) = t^n\sum_{k=0}^{n}(-1)^k\frac{f(b,t)g^{(k)}(b)-f(a,t)g^{(k)}(a)}{t^k} $$ We can compare the coefficients of this too. For $m\leq n$ contribution of $[g]_f(t)$ to $t^m$ is zero, so we get $$ (-1)^n \int_a^b f_m(x)g^{(n+1)}(x)dx = \sum_{k=n-m}^n (-1)^k \big( f_{k-n+m}(b)g^{(k)}(b)-f_{k-n+m}(a)g^{(k)}(a)\big) $$ For $m>n$, we start to see contributions from the other integral: $$ \begin{eqnarray*} \frac{m!}{(m-n-1)!}\int_a^bf_{m-n-1}(x)g(x)dx &=& \sum_{k=0}^n (-1)^k \big( f_{k-n+m}(b)g^{(k)}(b)-f_{k-n+m}(a)g^{(k)}(a)\big)\\ && + (-1)^{n+1} \int_a^b f_m(x)g^{(n+1)}(x)dx \end{eqnarray*} $$

Getting the Sums

There is one thing glaringly missing from these formulas, and that is the sums. The Euler-Maclaurin formula is supposed to approximate a sum by its integral, but the only sums we see here are the "extra" terms in the formula. To find these, let $f(x,t)$ be a Bernoulli-type function and $F(x,t)=f({x},t)$, where ${x}$ is the fractional part of $x$. If we set $\{g\}_F(t)=\int_u^{v+1} F(x,t)g(x)dx$, then we can split up the integrals to get $$ \{g\}_F(t) = \sum_{j=u}^{v}[T_jg]_f(t) $$ where $T_jg(x)=g(x+j)$, and we're assuming now that $a=0$ and $b=1$ in the definition of $[g]_f(x)$, and that $u,v\in\mathbb{Z}$. Using the results from above, we then get, after some rearranging, $$ \begin{eqnarray*} t^{n+1}\{g\}_F(t) - (-1)^{n+1}\{g^{(n+1)}\}_F(t) &=& t^n \sum_{k=0}^{n}(-1)^k\frac{f(1,t)g^{(k)}(v+1)-f(0,t)g^{(k)}(u)}{t^k} \\ && + t^n(f(1,t)-f(0,t))\sum_{k=0}^n\frac{(-1)^k}{t^k}\sum_{j=u+1}^vg^{(k)(j) \end{eqnarray*} $$
