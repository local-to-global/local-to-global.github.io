Stickelberger's Theorem

11/21/2016

If $K/\mathbb{Q}$ is a number field, then the class group $C(K)$ is a way to encode the complexity of the arithmetic in $K$. One place where this is apparent is in Kummer's partial result on Fermat's Last Theorem, which says that if $K=\mathbb{Q}(\zeta_p)$, and $p\nmid \# C(\mathbb{Q}(\zeta_p))$, then Fermat's Last Theorem holds for exponent $p$. Figuring out the shape and size of $C(K)$ typically leads to deep results.

Stickelberger's Theorem is a fairly general result on the divisibility properties of $C(\mathbb{Q}(\zeta_n))$. The group $G(\mathbb{Q}(\zeta_n)/\mathbb{Q})$ acts on the class group, and Stickelberger's Theorem then provides us with a recipie to kill the class group using this action.
