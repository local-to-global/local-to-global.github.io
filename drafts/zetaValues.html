Bernoulli Numbers and Values of the Riemann Zeta Function

11/21/2016

Bernoulli Numbers pop up in many contexts in number theory. They are defined to be the Taylor coefficients of the function $$ f(t) = \frac{t}{e^t-1} = \sum_{n=1}^{\infty}b_n\frac{t^n}{n!} $$ The reason why this function is important is because we can expand this out as a geometric series: $$ f(t) = te^{-t}\frac{1}{1-e^{-t}} = te^{-t} + te^{-2t} + te^{-3t}+ te^{-4t}+... $$ This sum can be used in many ways that we'll see. Related to this, we also can define the Bernoulli Polynomials as the Taylor coefficients of the following function (expanded in the variable $t$): $$ F(x,t) = \frac{te^{xt}}{e^t-1} = \sum_{n=1}B_n(x)\frac{t^n}{n!} $$ Note that $B_n(0)=b_n$. Now consider the value $F(x,t)(e^{(N+1)t}t-1)$ Using the formula $\frac{z^{N+1}-1}{z-1}=1+z+z^2+\cdots+z^N$ we get $$ \begin{eqnarray*} F(x,t)(e^{(N+1)t}-1) &=& te^{xt}\frac{e^{(N+1)t}-1}{e^t-1}\\ &=& te^{xt}(1+e^t+e^{2t}+\cdots +e^{Nt})\\ &=& t\sum_{k=0}^{N}e^{t(x+k)}\\ &=& \sum_{k=0}^{N}\sum_{n=0}^{\infty}(x+k)^n\frac{t^{n+1}}{n!}\\ &=& \sum_{n=0}^{\infty}(x^n + (x+1)^n+(x+2)^n+\cdots+(x+N)^n)\frac{t^{n+1}}{n!}\\ \end{eqnarray*} $$ So the Taylor Coefficients of this contain the sum of the first $N$ $n$th powers, when $x=0$. But we can write this differently as $$ \begin{eqnarray*} F(x,t)(e^{(N+1)t}-1) &=& \frac{te^{xt+(N+1)t}}{e^t-1} - \frac{te^{xt}}{e^t-1}\\ &=& F(x+N+1,t) - F(x,t)\\ &=& \sum_{n=1}^{\infty} (B_n(x+N+1)-B_n(x))\frac{t^n}{n!} \end{eqnarray*} $$ Plugging in $x=0$ and equating these Taylor coefficients then shows that $$ 1+2^n+3^n+4^n+\cdots+N^n = \frac{B_{n+1}(N+1)-B_{n+1}(1)}{(n+1)!} $$ And this followed directly from the nice properties of the function $F(x,t)$ and how we can rewrite it using geometric series.
