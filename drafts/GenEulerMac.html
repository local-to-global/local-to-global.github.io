Appell Sequences

An Appell Sequence of polynomials is a sequence $p_0(x),p_1(x),...$ so that the degree of $p_n(x)$ is $n$ and for $n>0$ we have $p'_n(x)=np_{n-1}(x)$. Equivalently, there is a sequence $c_0,c_1,...$ so that $p_n(x)=\sum_{k=0}^n\binom{n}{k}c_{n-k}x^k$. Furthermore, we have $p_n(x+y)=\sum_{k=0}^n\binom{n}{k}p_{n-k}(x)y^{k}$. In a lot of ways, these are polynomials that behave like $x^n$.

There is another characterization of Appell Squences in terms of analytic functions. If we have a function $P(x,t)$ that is analytic in $t$ with $\frac{d}{dx}P(x,t)=tP(x,t)$, then we can write $P(x,t)=\sum_{n=0}^{\infty}p_n(x)\frac{t^n}{n!}$, where $p_n(x)$ is an Appell sequence. An Appell sequence conversely defines such a function. Call this an Appell Function. Elementary differential equations shows that we can write $P(x,t)=c(t)e^{xt}$ where $c(x)=\sum_{n=0}^{\infty}c_n\frac{t^n}{n!}$, where $c_n$ is the sequence of coefficients associated to the Appell sequence.

If $P(x,t)=c(t)e^{xt}$ is an Appell function, and $a,b$ are fixed numbers, then define $\overline{P}(x,t)=\overline{c}(t)e^{xt}$ where $\overline{c}(t)=c(t/a)e^{bt/a}$. This is also an Appell function, and it is the function that satisfies $P(ax+b,t)=\overline{P}(x,at)$. The associated Appell sequence is given by $\overline{p}_n(x)=p_n(ax+b)/a^n$ and the addition formula for Appell sequences says $\overline{p}_n(x)=\frac{1}{a^n}\sum_{k=0}^n\binom{n}{k}f_{n-k}(b)a^kx^k$. Furthermore $\overline{c}_n=\frac{1}{a^n}\sum_{k=0}^n\binom{n}{k}c_kb^{n-k}$.

Some examples of Appell functions are:

  • The Traditional Exponent: $\exp(x,t)=e^{xt}$, whose Appell sequence is $\exp_n(x)=x^n$ and whose sequence is $e_0=1$ and $e_n=0$ for $n>0$. We have $\overline{\exp(x,t)}=e^{bt/a}e^{xt}$, $\overline{\exp}_n(x) = \frac{1}{a^n}\sum_{k=0}^n\binom{n}{k}b^{n-k}(ax)^k$ and $\overline{e}_n = b^n/a^n$.
  • The Bernoulli Function: $B(x,t) = \frac{te^{xt}}{e^t-1}$, whose Appell sequence is $B_n(x)$, the Bernoulli Polynomials, and whose number sequence are $B_n$, the Bernoulli Numbers. We have $\overline{B}(x,t)=\frac{B(x,t)}{a}\sum_{k=0}^{\infty}(e^{(b+k)t/a}- e^{(a+b+k)t/a})$. If $a=-1$ and $b=1$, then we get $\overline{B}(x,t)=B(x,t)$, which means that $\overline{B}_n(x)=B(x)$, and implies $B_n(1-x)=(-1)^nB_n(x)$.
  • The Euler Function: $E(x,t) = \frac{2e^{xt}}{e^t-1}$, whose Appell sequence are the Euler Polynomials $E_n(x)$. These satisfy identical properties to the Bernoulli Polynomials.

The Appell Operators

Fix an Appell Function $P(x,t)$ with Appell sequence $p_n(x)$. Given a function $g(x)$ and an interval $[a,b]$, we can define the linear operators: $$ \begin{eqnarray*} \mathcal{L}(g;t) &=& \int_a^bP(x,t)g(x)dx\\ \partial \mathcal{L}(g;t) &=& P(b,t)g(b) - P(a,t)g(a) \end{eqnarray*} $$ We can, additionally, define the functionals $$ \begin{eqnarray*} L_n(g) &=& \int_a^bp_n(x)g(x)dx\\ \partial L_n(g) &=& p_n(b)g(b) - p_n(a)g(a) \end{eqnarray*} $$ If the interval is not clear from notation, we'll put it as a superscript. We then get a power series representation for $\mathcal{L}^{a,b}(g;t)$ and $\partial \mathcal{L}^{a,b}(g;t)$ as $$ \begin{eqnarray*} \mathcal{L(g;t)} &=& \sum_{n=0}^{\infty} L_n(g) \frac{t^n}{n!}\\ \partial \mathcal{L}(g;t) &=& \sum_{n=0}^{\infty} \partial L_n(g) \frac{t^n}{n!} \end{eqnarray*} $$ Using integration by parts with the equation $\frac{d}{dx}P(x,t)=tP(x,t)$ we can get that $$ t\mathcal{L}(g;t) + \mathcal{L}(g';t) = \partial\mathcal{L}(g;t) $$ Comparing the coefficients in the corresponding power series gives $L_0(g')=\partial L_0(g)$, which amounts to the Fundamental Theorem of Calculus, but additionally $$ nL_{n-1}(g) + L_n(g') = \partial L_n(g) $$ If we iterate this process, for a total of $n+1$ times,then we get $$ \mathcal{L}(g;t)t^{n+1} - (-1)^{n+1}\mathcal{L}(g^{(n+1)};t) = \sum_{k=0}^{n}(-1)^k \partial \mathcal{L}(g^{(k)};t)t^{n-k} $$
